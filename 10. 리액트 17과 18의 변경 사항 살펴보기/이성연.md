### 10.2.4 자동 배치(Automatic Batching)
여러 상태를 하나의 리렌더링으로 묶어서 성능 향상

Promise, setTimeout같은 비동기이벤트에서도 자동배치 됨.

```jsx
function handleClick() {
    sleep(3000).then(()=> {
        setCount((c) => c + 1)
        setFlag(!f)
    })
}
```
